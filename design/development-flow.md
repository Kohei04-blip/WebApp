# 開発フロー


## 1. 課題整理

- Java,SQL,Gitなど、複数分野を並行して学ぶ中、復習の優先度が下がり、学んだ内容が定着しにくいことに悩んでいた。

---

## 2. 要件定義

- 学習内容を登録できる
- 分野ごとに整理できる
- ノード形式で知識をまとめられる
- 復習日を作成する
- クイズを登録できる
- 今日復習すべき内容を表示する
- クイズに回答して結果を保存できる
- 回答結果に応じて復習日を更新できる

---

## 3.画面一覧設計


3-1.　ホーム画面
  - 今日の復習件数
  - 優先度の高い復習一覧
  - 分野別の進捗

3-2. 分野別の一覧画面
  - Java
  - SQL
  - Git
  - Spring Boot 

3-3. ノード一覧画面
 - ツリー表示
 - 親子関係表示
 - ノード検索

3-4. ノード詳細画面　
  - 説明内容
  - 関連ノード
  - 登録ずみクイズ
  - 次回復習
      
3-5. クイズ画面
  - 問題表示
  - 回答表示
  - 自己評価ボタン
    - 思い出せた
    - なんとなく
    - 忘れた

3-6. 復習履歴画面
  - いつ復習したか
  - 苦手なノード
  - 放置されている単元
